# 📊 Сводная система мониторинга активности сотрудников (Desktop Activity Monitor)

Архитектурное решение для сбора, стриминга и анализа метрик рабочего времени. Система фиксирует перемещения курсора мыши, заголовки активных окон, а также асинхронно отслеживает события блокировки и разблокировки экрана Windows, отправляя данные через Apache Kafka в аналитическую СУБД ClickHouse.


---

## 🏗️ Архитектура и структура проекта





```text
 ├── ActivityMonitor/         # C# Консольное приложение (Windows-служба)
 │   ├── Program.cs           # Основной цикл опроса и логика приложения
 │   ├── EmployeeActivity.cs  # Взаимодействие с Win32 API и SystemEvents
 │   ├── settings.ini         # Локальные настройки (Kafka Host, интервалы)
 │   └── ActivityMonitor.csproj
 ├── clickhouse/              # Настройки и скрипты инициализации СУБД
 │   └── init-db.sql          # Скрипт автоматического создания таблиц и движка Kafka
 ├── images/                  # Графические материалы для документации (схемы, скриншоты)
 ├── docker-compose.yaml      # Инфраструктура в один клик (Kafka + Zookeeper + ClickHouse)
 └── README.md                # Этот файл

```
